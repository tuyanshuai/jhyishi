## 本地运行

```bash
# 使用Python内置服务器
python -m http.server 3000

# 或者直接打开index.html文件
```

## Vercel部署

1. 将代码推送到GitHub仓库
2. 在Vercel中导入项目
3. 自动部署完成

或者使用Vercel CLI：

```bash
# 安装Vercel CLI
npm i -g vercel

# 部署
vercel

# 部署到生产环境
vercel --prod
```
